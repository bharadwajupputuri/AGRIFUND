# 🌾 AGRIFUND — Peer-to-Peer Agricultural Financing Platform

AGRIFUND is a full-stack web platform that connects **farmers** and **investors**, enabling seamless peer-to-peer agricultural financing. The system helps farmers raise funds for their agricultural needs while allowing investors to support and earn returns from farming projects.

---

## 🚀 Features

* 👨‍🌾 **Farmer Portal**

  * Create and manage funding requests
  * Track loan status and repayments
  * View investor interest

* 💰 **Investor Portal**

  * Browse agricultural projects
  * Invest in farmers directly
  * Track portfolio and returns

* 🔄 **Real-Time Transactions**

  * Secure and efficient financial interactions
  * Concurrent data handling

* 📊 **Portfolio Management**

  * Investment tracking
  * Loan and interest calculations

* 🔐 **Authentication & Security**

  * User login & registration
  * Secure data handling

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/bharadwajupputuri/AGRIFUND.git
cd AGRIFUND
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Configure database

* Create a MySQL database
* Update database credentials in config file

### 4️⃣ Run the server

```bash
npm start
```

### 5️⃣ Open in browser

```
http://localhost:3000
```

## 📌 How It Works

1. Farmers create projects requiring funding
2. Investors browse and invest in projects
3. Platform manages loans, repayments, and interest
4. Both users can track activities via dashboards

Platforms like AGRIFUND aim to reduce dependency on traditional banking and improve financial accessibility for farmers, similar to modern agri-tech solutions that enhance transparency and funding access.
